Assignment 2

Choose either Option A or Option B (not both).

Option A — see the attached handout for details.

Option B (more challenging & exploratory): read the classic paper “Stanford Town” (Generative Agents: Interactive Simulacra of Human Behavior), which introduces an architecture combining large language models with memory, reflection, and planning to simulate realistic individual and group behaviors in a sandbox environment. If you choose Option B, you need to:

  1. Read and understand the paper.
  2. Prepare a presentation explaining the research.
  3. After this week's lab, before the deadline, try to find and run the original code of the paper. (It's fine if you cannot fully reproduce the simulation.)
  4. Submit a written reflection, describing at least three major challenges you faced and how you addressed them.

You may discuss with your group or consult an LLM if needed. Option B is closely related to what we'll implement at the end of this course — a social simulation using LLMs — so it gives you early hands-on experience.

Please note:

  • If you choose Option B, you do not need to complete Option A (it's either A or B).
  • Students choosing Option B must form a new temporary group (different from your previous one, and only for this assignment).
  • The code for the paper: https://github.com/joonspk-research/generative_agents
